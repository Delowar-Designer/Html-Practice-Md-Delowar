HTML (HyperText Markup Language) কোনো প্রোগ্রামিং ভাষা নয়, বরং এটি একটি মার্কআপ ভাষা যা ওয়েব পেজের কাঠামো (structure) তৈরি করতে ব্যবহৃত হয়, যেমন শিরোনাম, অনুচ্ছেদ, ছবি ইত্যাদি যুক্ত করা হয়, যা ব্রাউজার প্রদর্শন করে। এটি ওয়েব ডেভেলপমেন্টের ভিত্তি এবং এর মাধ্যমে ওয়েবসাইটের কন্টেন্ট সাজানো হয়।



পূর্ণরূপ: HyperText Markup Language (হাইপারটেক্সট মার্কআপ ল্যাঙ্গুয়েজ)। কাজ: ওয়েব পেজের বিভিন্ন উপাদান (যেমন টেক্সট, ছবি, লিঙ্ক) চিহ্নিত করা ও সাজানো। প্রকারভেদ: এটি কোনো প্রোগ্রামিং ভাষা না হলেও, ওয়েব ডিজাইনে একে মার্কআপ ভাষা হিসেবে বিবেচনা করা হয়।

উদ্ভাবক: টিম বার্নার্স-লি (Tim Berners-Lee)। HyperText Markup Language (HTML) is the standard coding language used to structure and present content on the World Wide Web, defining elements like headings, paragraphs, links, and images with tags (e.g.,) that browsers interpret to display web pages, forming the fundamental building block of every website alongside CSS (styling) and JavaScript (interactivity).
